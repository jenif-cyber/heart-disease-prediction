# ❤️ Heart Disease Prediction Project

This is a beginner machine learning project that predicts heart disease using patient medical data.

## 📊 Dataset
UCI Heart Disease Dataset

## 🧠 Steps
- Data loading
- Data cleaning
- Train-test split
- Feature scaling
- Logistic Regression model
- Evaluation

## 📈 Results
- Accuracy: 88.5%
- Confusion Matrix: [[25 4], [3 29]]

## 🎯 Goal
To learn basic machine learning workflow using real-world medical data.
